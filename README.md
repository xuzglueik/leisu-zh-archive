# leisu-zh-archive

这是一个用于归档和发布多个独立 HTML 页面的仓库。存放的页面内容以中文为主，涵盖不同主题与用途，不针对任何特定域名或网站。

## 项目简介

本仓库旨在长期保存并公开分享一些独立制作的 HTML 页面。这些页面可能包括信息展示、工具页面、静态文档等，均以单一 HTML 文件形式存在，便于直接访问、下载或离线使用。

## 目录说明

```
.
├── index.html          # 仓库主索引页面（可选）
├── pages/              # 存放各归档 HTML 页面
│   ├── example-page-1.html
│   ├── example-page-2.html
│   └── ...
└── README.md           # 本文件
```

- `pages/` 目录下存放所有归档的 HTML 页面文件。
- 每个页面均为独立完整文件，不依赖外部资源（或资源已内嵌）。
- 如有索引页面，会列出主要归档页面的链接和简介。

## 页面归档说明

- 所有页面均为静态 HTML，无后端依赖。
- 页面内容可能包含文本、图片（Base64 内嵌或相对路径引用）、样式、脚本等。
- 归档页面可能不定期更新或修正，但会尽量保持历史版本可追溯。
- 部分页面可能包含时间戳或版本标识，用于区分不同归档批次。
